Limelight Cinema

The website Limelight Cinema was a brief given throughout university, with goals to create a functioning website with PHP to display movies from a Database. A system was implemented so non-registered users could not access Adult Movies as it required a minimum age of 14. The pages provided on the website was the Home Page which gave users a brief understanding of what movies were on along with a Contact Form. The Adult Movies Page was provided which showed movies that were aged 14+, along with the Junior Movies Page which displayed child friendly movies. A Games Page was implemented which allowed users to complete a quiz, and a Login/Register Page which allowed users to login to their account and make bookings for specific movies. Admin Login could also be accessed through the Login/Register Page.

Key Features

Map Information

Displayed in the footer was a map location of where the cinema was located. This was done through the use of Google Map iFrame, which allowed a direct pin-point of the cinema on the map, which makes it easier for users to locate the movie theatre

PHP

The purpose of the website was to display movies, where users can book a time to go and watch. As certain movies are age restricted, users who wished to access the Adult Movies Page were hit with a barrier, where they had to create an account to verify they were aged 14 or over. This eliminated the chance of having underage viewers at a movie, through the use of PHP.

Database Functionality

The Database for Limelight Cinema was designed to have a stock control, which meant that only a certain number of tickets were available per movie. This meant when the stock had ran out, users would not be able to purchase any more tickets.
