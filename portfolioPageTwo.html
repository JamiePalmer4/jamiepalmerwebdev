Hi-Top Sneakers

The website Hi-Top Sneakers was a project brief which was handed to me during for a university project. This given piece of work incorporated my key skills that I had developed throughout the year and combined them into one final piece of work. The website featured a Home Page which was viewed as a Shop Page, where users who interacted with the website can have access to specific sneakers, where they can purchase the product through an integrated PayPal system. Other pages included an About Us Page, Contact Page, and a Register/Login Page which stored users details which made the process for purchasing a product easier.

Key Features

PayPal Integrated System

The brief required a payment system to be incorporated into the website for the Shop function. The PayPal system was integrated to allow users to purchase any of the stock available on the website. Users could add their item to a basket then proceed to the PayPal checkout.

Registration of New Users

In order for users to purchase stock, they firstly had to register an account with Hi-Top Sneakers. Details were stored securely in the Hi-Top Sneakers database. This allowed users to purchase products, and for users who wishes to purchase shoes later on have an easy access where they do not have to create a new account.

Database

The Database stores the records for all registered users on the website, along with the stock records. Access to the database can be found through the Admin Login form, which has a unique username and password in order to access both databases, and alter either of them.
